# Revenue Assistant

RevenueFlow Assistant Chrome extension and Cloud Sync backend.

## Folders

- `RevenueFlow_Assistant` - Chrome Extension MV3 source.
- `RevenueFlow_Cloud_Sync` - backend/cloud sync scaffold for PayPal API, CSV fallback, and future provider adapters.

## Safety

Do not commit `.env`, logs, output data, packaged zip files, or customer credentials.
