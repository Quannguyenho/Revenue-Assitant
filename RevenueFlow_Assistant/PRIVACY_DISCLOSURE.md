# Chrome Web Store Privacy Disclosure

Use these answers in the Privacy practices tab and keep them aligned with the published privacy policy.

## Data handled

- Personally identifiable information: Gmail address, customer names, customer email addresses.
- Financial and payment information: payment amount, provider, order/reference number, transaction/profile identifiers.
- Personal communications: payment-related Gmail message content.
- User activity: only extension workflow actions and locally generated history; no analytics or cross-site tracking.
- Website content: only the currently open Gmail message when the user explicitly invokes the manual fallback.

## Purposes

- App functionality only.
- Payment-email extraction, duplicate review, user-selected CSV export, and user-configured Google Sheets writes.

## Data practices

- Data is not sold.
- Data is not used for advertising.
- Data is not used for creditworthiness or lending.
- Gmail content and parsed records are not transmitted to a Netbase JSC server in this release.
- Settings and up to 50 history records are stored locally in the Chrome profile.
- Google data is transferred only to Google APIs as required for Gmail reads and Sheets operations.

## Limited Use affirmation

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
