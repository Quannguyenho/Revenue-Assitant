# RevenueFlow Assistant v3.3.0 Settings UX

This build optimizes the Settings panel for non-technical users.

## Main changes
- Settings opens as a setup hub instead of a long technical form.
- Google Sheet setup is first and easier to understand.
- Payment provider rules are shown as clean cards with enable/disable toggles.
- Product mapping by amount remains visible because users need it often.
- JSON rule editing, regex, and legacy source rules are moved into advanced collapsible sections.
- Automation, exchange rate, invoice, appearance, and history are separated into clearer sections.

## Preserved
- Gmail scan logic.
- Google Sheet write logic.
- Payment rules engine.
- Existing DOM IDs and JavaScript event handlers.
- Chrome OAuth setup flow.
