# Chrome Web Store Listing

## Product name

RevenueFlow Assistant

## Single purpose

RevenueFlow Assistant scans a user's Gmail mailbox for payment notifications, lets the user review structured payment records, and writes approved records to a user-selected Google Sheet.

## Short description

Scan payment emails, review PayPal and Stripe records, and save approved revenue data to Google Sheets.

## Detailed description

RevenueFlow Assistant gives small businesses a clear payment reconciliation workflow inside Chrome:

- Connect a Gmail account with Google OAuth.
- Scan for supported payment notifications without opening each email.
- Review extracted customer, order, amount, provider, and transaction fields.
- Detect possible duplicates before saving.
- Filter PayPal and Stripe records.
- Automatically create a private RevenueFlow Sheet in the connected Google account when needed, then write approved rows.
- Export local payment history as CSV.

RevenueFlow requests read-only Gmail access. It cannot send, delete, move, or modify messages through the Gmail API. Google Sheets access is used for the spreadsheet selected by the user. Gmail passwords are never requested or stored.
